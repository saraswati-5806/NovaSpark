# 🚀 NovaSpark — Premium IT Solutions Landing Page

NovaSpark is a modern, responsive, and visually engaging SaaS-style landing page developed using HTML, CSS, and JavaScript.
The project focuses on delivering a professional UI/UX experience with smooth animations, glassmorphism-inspired components, responsive layouts, and modern digital branding aesthetics.

Designed and developed by **Saraswati Panigrahi**.

# 📌 Project Overview

NovaSpark is a fictional premium IT solutions and digital services company landing page created to showcase modern frontend development practices.

The project emphasizes:

* Responsive web design
* Professional UI/UX principles
* Structured layouts
* Smooth animations
* Clean typography
* Modern color systems
* Interactive user experience

This project was built as part of frontend web development practice and portfolio enhancement.

---

# ✨ Features

## ✅ Responsive Design

The entire website adapts seamlessly across:

* Desktop
* Tablet
* Mobile devices

Using CSS Grid, Flexbox, and media queries.
